# Инструкция по деплою на Render

Так как Railway запрещает использование ботов, мы будем использовать **Render (render.com)**. Это отличная альтернатива с бесплатным тарифом.

## Шаги для деплоя

1.  **Зарегистрируйтесь** на [Render.com](https://render.com/), используя ваш GitHub аккаунт.
2.  **Создайте новый Blueprint (Blueprint Instance)**:
    *   В панели управления Render нажмите кнопку `New +`.
    *   Выберите `Blueprint`.
    *   Подключите этот репозиторий (`telegram-finance-bot`).
3.  **Настройка**:
    *   Render автоматически обнаружит файл `render.yaml` из этого репозитория.
    *   Вам будет предложено ввести значения для переменных окружения, которые помечены как `sync: false` (они приватные и не хранятся в коде):
        *   `TELEGRAM_BOT_TOKEN`: Токен вашего бота от @BotFather.
        *   `OPENAI_API_KEY`: Ваш ключ API от OpenAI.
4.  **Запуск**:
    *   Нажмите `Apply`.
    *   Render создаст базу данных PostgreSQL и Worker сервис для вашего бота.
    *   Бот автоматически запустится и подключится к базе данных.

## Важно

*   Если вы используете бесплатную базу данных на Render, она будет удалена через 90 дней. Для продакшена рекомендуется обновиться до платного тарифа (Starter) или использовать внешний сервис базы данных (например, Neon.tech), изменив переменную `POSTGRES_CONNECTION_STRING`.
